# UKrAiNeCOLL - UA-CYBER Verifier Prank

A visual-rich, interactive single-page website styled like a futuristic "Ukrainian hacker scam" portal. It challenges visitors to "verify their Ukrainian citizenship" by completing a financial card details form, then plays a mock hacking sequence and reveals that their "stolen funds" were donated to volunteers for buying FPV-drones for the Ukrainian Armed Forces (AFU).

## ⚠️ Important Disclaimer
**This is a client-side joke/prank website.** 
Confidential data entered into the form (card number, expiry, CVV) is processed strictly locally inside the visitor's browser. It is **never** sent to any external server or saved anywhere. The site is designed solely for entertainment and visual demonstration purposes.

- **Patriot Matrix Rain**: Background canvas animation drawing falling Ukrainian/Cyrillic characters in neon blue and gold-yellow columns.

- **Interactive Cossack Card**: A 3D-flipping credit card visualization syncing input details and rotating on CVV focus.
- **Typing Log Terminal**: Satirical console logs conducting automated patriotism and DNA/Borscht diagnostics.
- **Progress exploits**: Retro hacking progress simulator with simulated status logs.
